Esse relógio foi feito por Vinícius Peruzzi.
Na falta de um relógio verdadeiro para uma boa partida de xadrez, e inclusive na falta de bons aplicativos para essa finalidade, e principalmente que sejam gratuitos, esse site foi desenvolvido para que possa ser acessador a partir de qualquer lugar ou dispositivo.
Acompanhe esse e outros projetos visitando o meu GitHub ou então a minha página pessoal (Links no cabeçalho dessa página).

Para utilizar o relógio, você deve setar as configurações de cada um dos relógios, utulizando os campos específicos para isso. Depois de setadas as informações a contagem pode começar, o primeiro jogador deve fazer o movimento e teclar no botão click. Com isso a contagem começa e a cada click vira o lado da contagem.
